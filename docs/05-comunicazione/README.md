Questa sezione documenta gli strumenti di comunicazione della Suite Clinica, includendo sia i flussi interni sia le integrazioni esterne collegate alla messaggistica e al lifecycle clienti.

## Panoramica dei Flussi

Il sistema di comunicazione si integra trasversalmente con gli altri moduli:
- **Team**: Per il targeting degli annunci interni per dipartimento.
- **Clienti**: Per l'invio di messaggi WhatsApp e il tracking del lifecycle tramite Respond.io.
- **Task**: Per l'invio di notifiche push all'assegnazione di un nuovo compito.
