Manuale Check

Guida completa per nutrizionisti, coach e psicologi alla gestione dei check clienti

Introduzione

Il sistema Check 2.0 permette di monitorare i progressi dei tuoi clienti attraverso questionari periodici. Questa guida ti aiuterà a gestire efficacemente i check ricevuti.

Cosa Puoi Fare

💡 Funzionalità Principali
  • Visualizzare i check compilati dai tuoi clienti
  • Confermare la lettura dei check ricevuti
  • Generare link per nuovi check
  • Monitorare progressi
  • Ricevere notifiche automatiche via email

Tipi di Check

Tipo Descrizione Frequenza
Check Regolare Per tutti i clienti Settimanale
Check DCA Per clienti con disturbi del comportamento alimentare (tipo 3) Settimanale

1. Visualizzare Check nella Scheda Cliente

1.1 Accedere alla Sezione Check

Ogni cliente ha una sezione dedicata dove puoi vedere tutti i check compilati.

📍 Procedura
  1. Vai alla lista clienti: Menu → Clienti
  2. Cerca e clicca sul cliente desiderato
  3. Nella scheda cliente, clicca sul tab Check 2.0
  4. Vedrai l'elenco completo di tutti i check del cliente

1.2 Cosa Puoi Vedere

Nella sezione Check 2.0 troverai:

  • Lista completa check: tutti i check compilati dal cliente ordinati per data
  • Tipo check: badge colorato che indica Regolare o DCA
  • Data e ora: quando il check è stato compilato
  • Azioni: pulsanti per visualizzare dettagli

1.3 Visualizzare Dettagli Check

🔍 Come Vedere i Dettagli
  1. Nella lista check, clicca sul check che ti interessa
  2. Si aprirà un modal con tutti i dettagli:
    • Risposte alle domande
    • Valutazioni numeriche (es. soddisfazione nutrizionista, psicologa, coach)
    • Feedback testuali del cliente
    • Foto di progresso (fronte, retro, laterale)
    • Peso e misurazioni
  3. Puoi navigare tra le varie sezioni del modal
  4. Per chiudere, clicca sulla X o fuori dal modal

2. Check da Leggere e Conferma Lettura

2.1 Notifica Badge nel Menu

Quando ricevi nuovi check da leggere, vedrai un badge rosso con il numero nel menu principale accanto alla voce "Check".

🔔 Notifiche Automatiche
  • Badge nel menu: indica il numero di check non ancora letti
  • Email automatica: ricevi una notifica via email quando un tuo cliente compila un check

2.2 Accedere alla Pagina "Check da Leggere"

📍 Procedura
  1. Nel menu principale, clicca su Check (vedrai il badge rosso se hai check da leggere)
  2. Seleziona Check da Leggere
  3. Vedrai l'elenco di tutti i check in attesa di conferma lettura

2.3 Cosa Vedi nella Pagina

La pagina "Check da Leggere" mostra solo i check dei tuoi clienti che non hai ancora confermato:

  • Data/Ora: quando il check è stato compilato
  • Cliente: nome del cliente e programma attuale
  • Azioni:
    • Vai alla scheda cliente: apre la scheda completa del cliente
    • Conferma Lettura: conferma che hai letto il check
⏳ In Attesa vs Confermato

I check rimangono nella lista "da leggere" finché non clicchi su "Conferma Lettura". Una volta confermato:

  • Il check sparisce dalla lista "Check da Leggere"
  • Il badge nel menu si aggiorna automaticamente
  • Il check rimane comunque visibile nella scheda cliente con badge ✅ Letto

2.4 Confermare la Lettura

✅ Come Confermare
  1. Clicca sul check per visualizzarne i dettagli nel modal
  2. Leggi attentamente le risposte del cliente
  3. Chiudi il modal
  4. Clicca sul pulsante Conferma

3. Generare Link per Nuovi Check

3.1 Quando Generare un Link

Genera un link check quando:

  • ✅ Il cliente è nuovo e non ha ancora ricevuto il link
  • ✅ Il cliente ha perso il link originale
  • ✅ Vuoi inviare un nuovo check (regolare o DCA)
  • ✅ Il link precedente è scaduto

3.2 Accesso alla Funzione

📍 Procedura
  1. Vai alla scheda del cliente: Menu → Clienti → [Nome Cliente]
  2. Apri il tab Check 2.0
  3. Nella sezione "check settimanale", vedrai i pulsanti per generare i link

3.3 Tipi di Link Disponibili

Tipo Check Descrizione Pulsante
Check Regolare Per tutti i clienti Genera Link Regolare
Check DCA Per clienti con disturbi del comportamento alimentare (tipo 3) Genera Link DCA

3.4 Cosa Succede Dopo

Una volta generato il link:

  • Il professionista deve mandare il link al cliente
  • Il sistema invia automaticamente una email a te e agli altri professionisti associati quando il check viene compilato
  • Tu ricevi una notifica via email e il badge nel menu si aggiorna
  • Il check appare nella lista "Check da Leggere"
⚠️ Importante

Ogni link è univoco e personale per quel cliente. Non condividere link tra clienti diversi. Se un cliente perde il link, genera un nuovo link specifico per lui.
